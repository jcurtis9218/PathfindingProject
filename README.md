# Pathfinding Project
[Video Link](https://drive.google.com/file/d/1Ru1Aq1peoAsCXOVcNX5_lnz-1fiW_e73/view?usp=sharing)
An implementation of A\* pathfinding on a randomly generated tilemap.
White tiles: Normal
Sand tiles: Harder to walk on, but still safe
Red circle tiles: Risky. Walkable, but considered dangerous.
Black tiles: Impassable (perhaps walls or pitfalls?)
